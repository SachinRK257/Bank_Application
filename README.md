# 🏦 Bank Application

A Java-based banking application designed to manage customer accounts and perform essential banking operations such as deposits, withdrawals, transfers, and viewing transaction history. The project uses **JDBC** for database connectivity and **MySQL** for data storage.

---

## 📘 Overview

This Bank Application automates common banking activities and ensures accurate and secure handling of customer financial records.  
It demonstrates practical implementation of **Core Java**, **OOP concepts**, **JDBC**, and **MySQL CRUD operations**.

---

## 🚀 Features

### 🔐 User & Admin
- Admin authentication  
- Manage customer accounts

### 👤 Account Management
- Create new bank accounts  
- Update customer details  
- View account information  
- Close/Delete account  

### 💰 Banking Operations
- Deposit money  
- Withdraw money  
- Transfer funds between accounts  
- Check balance in real-time  

### 📄 Transaction History
- Logs deposits, withdrawals, and transfers  
- Allows user/admin to view transaction records

---

## 🛠️ Tech Stack

- **Java (Core + OOP)**  
- **JDBC**  
- **MySQL Database**  
- **Eclipse / IntelliJ / VS Code**  
- **SQL Queries**
